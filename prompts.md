## 【提示词 1/8】项目总览与技术栈定义

```
# 任务：项目初始化与整体框架搭建

## 一、页面目标
搭建宪法序言交互式教学网页的整体框架，包含
4个核心模块入口（序言展示、版本迭代、知识
闯关、共创序言），确保项目结构清晰、技术栈
统一、可扩展性强。

## 二、页面结构
创建以下文件结构：

/
├── index.html # 首页（四个模块入口）
├── preamble.html # 序言展示页
├── versions.html # 版本迭代页
├── quiz.html # 知识闯关页
├── co-create.html # 共创页
├── css/
│   ├── variables.css # CSS变量（颜色、字体、间距）
│   ├── base.css # 基础样式、字体引入、重置样式
│   └── components.css # 公共组件样式
├── js/
│   └── main.js # 公共逻辑
├── data/
│   ├── preamble.json # 宪法序言数据
│   └── questions.json # 题库数据
└── assets/
    └── images/ # 图片资源

## 三、视觉约束
- 技术栈：原生 HTML5 + CSS3 + JavaScript（ES6+），禁止引入 jQuery、Bootstrap
- 样式方案：CSS变量 + CSS Grid/Flexbox，禁止使用 Tailwind CSS（本项目追求精细控制）
- 配色：
  - 主色：#8B1A1A（深红色，体现庄重）
  - 辅助色：#D4AF37（金色，用于强调）
  - 背景：#FAFAF8（米白色，非纯白）
  - 文字：#1A1A1A（深灰，非纯黑）
  - 禁止：蓝紫渐变、大面积渐变背景
- 字体：
  - Display字体（标题）：'Noto Serif SC', serif
  - Body字体（正文）：'Noto Sans SC', sans-serif
  - 禁止：全站使用同一字体、使用 Inter 字体
- 禁止项：
  - 禁止 emoji 代替图标
  - 禁止所有内容居中布局
  - 禁止"图标+标题+描述"三栏卡片模板
  - 禁止悬停上浮动画（改用其他微交互）

## 四、修改范围
- 允许创建上述所有文件
- 允许引入 Google Fonts（Noto Serif SC、Noto Sans SC）
- 禁止引入任何第三方 CSS 框架
- 禁止引入任何第三方 JS 库（后续按需单独引入）

## 五、验收标准
1. 打开 index.html，页面正常显示，无控制台错误
2. CSS变量在 variables.css 中定义完整（颜色、字体、间距、阴影）
3. 字体正确加载，标题使用衬线字体，正文使用无衬线字体
4. 文件结构符合上述规范
5. 使用 `python -m http.server 8080` 或 Live Server 启动本地服务器测试
```

## 【提示词 2/8】CSS基础层与设计规范

```
# 任务：CSS变量与基础样式定义

## 一、页面目标
建立项目的设计系统基础，包含CSS变量定义、
重置样式、字体引入、通用工具类，确保后续开
发风格统一、易于维护。

## 二、页面结构
在 css/variables.css 中定义：
- 颜色变量（主色、辅助色、背景色、文字色、状态色）

- 间距变量（4px基准，生成4/8/12/16/24/32/48px）
- 阴影变量（轻/中/重三档）
- 过渡变量（快/中/慢三档）

在 css/base.css 中定义：
- CSS重置（使用现代重置方案，非 reset.css）
- 字体引入（Google Fonts链接）
- 根元素样式（html、body）
- 通用工具类（.sr-only、.container、.text-display、.text-body等）

## 三、视觉约束
- 颜色变量命名：--color-primary、--color-secondary、--color-bg、--color-text、--color-muted、--color-accent
- 字号层级：--text-xs(12px) → --text-sm(14px) → --text-base(16px) → --text-lg(18px) → --text-xl(24px) → --text-2xl(32px) → --text-3xl(48px)
- 间距遵循4px网格系统
- 禁止在组件样式中直接写颜色值，必须使用CSS变量
- 禁止使用 !important（除工具类外）

## 四、修改范围
- 只允许修改 css/variables.css 和 css/base.css
- 不允许修改任何HTML文件
- 不允许引入外部CSS文件

## 五、验收标准
1. variables.css 中所有变量定义完整，无遗漏
2. base.css 正确引入 Google Fonts
3. 在任意HTML中测试：`var(--color-primary)` 正确生效
4. 页面默认字体为 Noto Sans SC，标题类使用 Noto Serif SC
5. 无控制台CSS警告或错误
```

```
- 允许修改 js/main.js
- 允许修改 index.html（添加导航栏和页脚测试）
- 不允许修改 css/variables.css 和 css/base.css

## 五、验收标准
1. 导航栏在桌面端正确显示四个链接
2. 窗口宽度<768px时，导航栏变为汉堡菜单
3. 按钮hover状态过渡流畅
4. 页脚固定在页面底部
5. 所有组件在不同页面可复用
6. 无控制台JS错误
```
